Computer Organization and Design



Syllabus

Basic operational concepts, CPU structure, Arithmetic, Memory hierarchy, Input Output interfacing, Performance analysis, Design


  Module I
15%
Basic Structure of computers - functional units - Historical Perspective -Basic operational concepts - bus structures, Measuring performance: evaluating, comparing and summarizing performance

  Module II
15%
Memory locations and addresses - memory operations - instructions and instruction sequencing ,Instruction sets- RISC and CISC paradigms, Addressing modes
FIRST INTERNAL EXAMINATION

  Module III
15%
Computer arithmetic - Signed and unsigned numbers - Addition and subtraction - Logical operations - Constructing an ALU Multiplication and division - faster versions of multiplicationfloating point representation and arithmetic

  Module IV
15%
The processor: Building a data path - Simple and multi-cycle implementations - Microprogramming - Exceptions
SECOND INTERNAL EXAMINATION

  Module V
20%
Introduction to pipelining-pipeline Hazards, Memory hierarchy Caches - Cache performance - Virtual memory - Common framework for memory hierarchies

  Module VI
20%
Input/output - I/O performance measures - I/O techniques interrupts, polling, DMA; Synchronous vs. Asynchronous I/O; Controllers. Types and characteristics of I/O devices - Buses Interfaces in I/O devices - Design of an I/O system
END SEMESTER EXAM


Course Objectives
  • To lay the foundation for the study of hardware organization of digital computers.
  • To impart the knowledge on interplay between various building blocks of computer


Expected Outcome
  • The students will gain general idea about the functional aspects of each building blocks in computer design

QUESTION PAPER PATTERN (End semester exam)

Part A: 8 compulsory questions.
One question from each module of Module I - IV; and two each from Module V & VI.
Student has to answer all questions. (8 x5)=40
Part B: 3 questions uniformly covering modules I & II
Student has to answer any 2 from the 3 questions: (2 x 10) =20
Each question can have maximum of 4 sub questions(a,b,c,d), if needed.
Part C: 3 questions uniformly covering modules III & IV
Student has to answer any 2 from the 3 questions: (2 x 10) =20
Each question can have maximum of 4 sub questions(a,b,c,d), if needed.
Part D: 3 questions uniformly covering modules V & VI
Student has to answer any 2 questions from the 3 questions: (2 x 10) =20
Each question can have maximum of 4 sub questions(a,b,c,d), if needed.
